# My Portfolio

This is my personal portfolio website built with Vite and React.

## Pages

- **Home**: Welcome page
- **Games**: Showcase of games I've created
- **Products**: Showcase of products I've developed
- **About Me**: Information about myself

## Getting Started

1. Install dependencies: `npm install`
2. Start the development server: `npm run dev`
3. Open your browser to `http://localhost:5173`

## Build

To build for production: `npm un build`

## Technologies Used

- React
- TypeScript
- Vite
- React Router
